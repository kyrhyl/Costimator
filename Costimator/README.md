# Costimator

DPWH Integrated Cost Estimation System (Quantity Takeoff + DUPA/UPA + BOQ Management).

## Quick Start
```bash
cd Costimator
npm install
npm run dev
```

## Tests
```bash
npm test
```

## Environment
Create `Costimator/.env.local` with your MongoDB connection:
```
MONGODB_URI=your_mongodb_connection_string
```
